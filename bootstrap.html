waarom ik dit desgin gekozen heb

Mijn kleurenschema wilde ik simpel houden, omdat ik vind dat hoe simpeler een website is, hoe netter hij eruitziet.
Ik wilde dat mijn website niet te groot zou worden, en daarom heb ik ervoor gekozen om met linkjes te werken in plaats van een scroll-website. Ik vind dat je zeker een mooie scroll-website kunt maken, maar ik denk dat ik zelf een mooiere website kon maken met linkjes. Daarnaast werk ik daar ook altijd mee.
Mijn sitemap was als volgt: je hebt de main page, en vanuit die main page kun je naar de andere pagina’s. Ik wilde misschien ook nog een projectpagina toevoegen, met foto’s waarin de foto's zelf als linkjes fungeren, maar dat is allemaal extra werk voor ongeveer hetzelfde design.
Ik had niet echt screenshots van hoe mijn website eruit zou komen te zien — ik heb het gemaakt zoals ik op dat moment vond dat het eruit moest zien.
